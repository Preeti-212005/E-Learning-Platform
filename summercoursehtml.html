Summer Course on HTML

Master HTML this summer and kickstart your web development journey!

Course Overview

Our Summer Course on HTML is perfect for beginners and anyone looking to enhance their web development skills. You will learn to create responsive and accessible web pages using HTML.

Course Schedule

Duration: 6 weeks

Start Date: June 1, 2024

End Date: July 13, 2024

Classes: Saturdays and Sundays, 10 AM - 1 PM

Syllabus Highlights:

  • Introduction to HTML and its structure
  • Basic tags and elements
  • Text formatting and images
  • Links and navigation
  • Forms and input elements
  • HTML5 features
  • Best practices for web accessibility
  • Final project to create a personal webpage

By the end of this course, you will be equipped with the skills to build your own websites!
