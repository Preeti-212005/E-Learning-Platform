HTML Notes

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It describes the structure of web pages using markup.

Basic Structure of an HTML Document

An HTML document has a specific structure. Here is a simple example:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document Title</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

HTML Elements

HTML is made up of elements, which are represented by tags. Common tags include:

  • <h1> to <h6>: Heading tags.
  • <p>: Paragraph tag.
  • <a>: Anchor tag for links.
  • <img>: Image tag.
  • <div>: Division or section.
  • <span>: Inline container.

Attributes

HTML elements can have attributes that provide additional information. Attributes are always specified in the opening tag:

<a href="https://www.example.com">Visit Example</a>

In this example, href is an attribute of the <a> tag.

Lists

HTML supports ordered and unordered lists:

<ul> 
    <li>Item 1</li> 
    <li>Item 2</li> 
</ul>

<ol> 
    <li>First item</li> 
    <li>Second item</li> 
</ol>

Semantic HTML

Semantic HTML refers to the use of HTML markup that conveys meaning about the content. Examples include:

  • <header>: Represents introductory content.
  • <nav>: Represents navigation links.
  • <main>: Represents the main content of a document.
  • <article>: Represents a self-contained piece of content.
  • <footer>: Represents footer content.

Conclusion

HTML is the foundation of web development, providing structure and meaning to web content. Mastering HTML is essential for anyone looking to build websites.
