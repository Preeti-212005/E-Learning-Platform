Python Notes

Introduction to Python

Python is a high-level, interpreted programming language known for its easy readability and dynamic typing. It is widely used for web development, data analysis, artificial intelligence, scientific computing, and more.

Basic Syntax

    print(f"Hello, {name}!")
greet("World")

Data Types

Python has various built-in data types:

  • Integers: Whole numbers, e.g., 5, -10
  • Floats: Decimal numbers, e.g., 3.14, -0.001
  • Strings: Text data, e.g., "Hello, Python!"
  • Lists: Ordered collections, e.g., [1, 2, 3, 4]
  • Dictionaries: Key-value pairs, e.g., {"name": "Alice", "age": 25}

Control Flow

Python uses if statements to control the flow of execution:

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Loops

Python supports both for loops and while loops:

for i in range(5):
    print(i)

Functions

Functions in Python are defined using the def keyword:

def add(x, y):
    return x + y
result = add(5, 3)
print(result)  # Output: 8

Conclusion

Python is a versatile language that is beginner-friendly and has a vast ecosystem of libraries and frameworks, making it a great choice for various programming tasks.
